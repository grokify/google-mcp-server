## [v0.1.0] - 2026-02-09

### Highlights

- Initial release: Read-only MCP server for Google Slides presentations

### Added

- `get_presentation` tool: retrieve presentation metadata (title, slide count, locale, revision) ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
- `list_slides` tool: enumerate slides with titles and element counts ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
- `get_slide` tool: get content and elements for a specific slide by index or object ID ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
- `get_slide_notes` tool: get speaker notes for a specific slide ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
- `get_presentation_content` tool: aggregate all slides' text and images in one call ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
- Google service account authentication via `-credentials` flag ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
- goauth CredentialsSet authentication via `-goauth-credentials-file` and `-goauth-credentials-account` flags ([`638848d`](https://github.com/grokify/google-mcp-server/commit/638848d))
